# Changelog

## 0.1.0

- Initial Warnyin Agents MVP.
- Adds a VS Code activity view with Warnyin Paper Office pixel canvas.
- Adds locked install state, installer actions, Warnyin command forms, raw command replay, and command history.
- Adds workspace-scoped office layout persistence with edit, save, reset, import, export, and presets.
- Adds per-role palette persistence.
- Adds Warnyin workflow intelligence summary from local stage artifacts.
- Adds complete Warnyin slash command coverage, including `explore`, `next`, and role-aware `install-skill`.
- Adds persisted Claude transcript cursors/snapshots for reduced rescanning, main/sub-agent status, source, elapsed time, failed tool results, blocked/failed notifications, and token usage.
- Adds manifest-backed pixel furniture assets plus per-role body/hair/accessory character customization.
- Adds furniture placement/deletion, undo/redo, keyboard layout shortcuts, import preview, speech bubbles, command previews, saved prompts, validation hints, and archived topic visibility.
- Adds richer workflow metrics and visualization for dependency edges, build waves, review panel, verify loop, fix counts, and ship archive targets.
- Adds user-created custom layout presets, multi-select/bulk layout movement, and optional webview sound cues.
- Adds domain unit tests, Playwright webview canvas smoke tests, marketplace metadata, and VS Code packaging script.
